# PyRPS

 - Utility for testing RPS

## How to start?

- Install `requirements.txt`
- Run command `python PyRPS.py -url url -time time`

## Flags

- `-time` How many seconds will be tested. Default is 5 seconds.
- `-url` URL to be tested.
